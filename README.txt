The present CD contains the master's thesis

    On the scope of McMillan's formula

by Jan Berges. All results presented therein may be recalculated by means of the
provided source codes, namely the software package 'ebmb' and the Python scripts
contained in the directory 'thesis/scripts'. To be able to run the latter, it is
necessary that the further be installed and made accessible, e.g. by setting the
environmental variables 'PATH' and 'PYTHONPATH'.
